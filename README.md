# CCL-binarycode
Real time energy waste detection using computer vision
# The problem 
Standby energy, also known as "vampire power" is when electronic devices use up energy without being used.\
This costs the world $19 billion annually.\
Nobody notices, Nobody Acts. Until now.
# What it does
* uses your laptop camera to detect unattended devices
* alerts you with live energy waste data
# Features 
- Real time object and person detection
- Live energy, carbon and cost calculations
- Multi device tracking
- Instant red screen alert
# Tech Stack
- Python
- YOLOWorld (object detection)
- OpenCV
- Streamlit
